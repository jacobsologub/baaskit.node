BaaSKit.Node
============

A Backend as a Service Kit for Node.js

## Requirements

* Linux Server
* Node.js
* MongoDB

## Setup

Create an app:

    node baaskit -c 'ApplicationName'

Start the server:

    node baaskit -p 8810
